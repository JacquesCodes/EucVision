# EucVision

UAV-based remote sensing study of juvenile *Eucalyptus* plantation trials, developed as part of an MSc Forestry thesis at Stellenbosch University.

![EucVision deliverables flowchart](Diagrams/NewFlowchart-Deliverables%20.drawio.svg)

## Site

Trial site: IMPACT OAL, near Stellenbosch, South Africa ([eucxylo.sun.ac.za](https://eucxylo.sun.ac.za)).

Five *Eucalyptus* species/clones are monitored — Grandis, Grandis clone, Urophylla, Cloeziana, and Cladocalyx — across four planting spacings (1×1 m, 2×2 m, 3×3 m, 5×5 m) with three plot replicates each. Repeated DJI Matrice 3D UAV flights provide the imagery and point cloud data underlying the analysis.
